# 1. Make “Median Salary” the Hero Stat

## Goal

Create a clear focal point instead of four equal-weight metrics.

## Tasks

### A. Change layout structure

**Before**

```
[ A ][ B ][ C ][ D ]
```

**After**

```
[   MEDIAN SALARY (large)   ]
[ A ][ B ][ C ]
```

---

### B. Implement in Next.js (React + Tailwind example)

```jsx
<div className="grid gap-6">
  {/* Hero Stat */}
  <div className="bg-white rounded-2xl p-8 shadow-sm border">
    <p className="text-sm text-gray-500">Median Salary</p>
    <h2 className="text-4xl font-semibold tracking-tight">$101,020</h2>
  </div>

  {/* Secondary Stats */}
  <div className="grid grid-cols-3 gap-4">
    {stats.map((stat) => (
      <div key={stat.label} className="bg-white rounded-xl p-5 border">
        <p className="text-xs text-gray-500">{stat.label}</p>
        <p className="text-lg font-medium">{stat.value}</p>
      </div>
    ))}
  </div>
</div>
```
