# PT URL Discovery Agent — Claude Code Instructions

## Your Job
You are running a URL discovery pipeline for DPT (Doctor of Physical Therapy) programs.
Goal: For each PT program in the input CSV, find the URL of its **Financial Fact Sheet** page using Google Search via the Serper API.

## Setup First
```bash
pip install requests pandas python-dotenv --break-system-packages
```

Create a `.env` file in this directory:
```
SERPER_API_KEY=your_key_here
```

Get a free key at https://serper.dev (2,500 free searches — no credit card needed)

## Run Order
1. `python 01_load_programs.py`        — loads school list into pt_programs.csv
2. `python 02_discover_urls.py`        — searches for financial fact sheet URLs (resumable)
3. `python 03_export_review.py`        — exports not-found / low-confidence rows for manual review

## Input
Your school list CSV should be named `input_programs.csv` in this directory.
Required columns: `program_id`, `school_name`, `city`, `state`
Optional but helpful: `program_url` (the program's main website — improves search accuracy)

## Output
`output/pt_programs.csv` — master file, never deleted, always appended/upserted
`output/pt_review.csv`   — rows needing manual review (generated by step 3)

## Important Rules
- NEVER delete output/pt_programs.csv — always upsert
- If a run is interrupted, just re-run 02_discover_urls.py — it skips `url_found` rows
- Rate limit: 2-3 second delay between searches, 60s backoff on 429
- Max 3 search attempts per school before marking `search_exhausted`
